## Description

Spellbook Agent is an interactive AI-powered web app for generating fantastical spells based on user-selected magic type, difficulty, and mood. It features a custom Gradio user interface, stylish themes, and creative text generation powered by Hugging Face's hosted LLMs.

- Fully free to use and remix.
- No login required for spell generation.
- Optimized for instant response and easy sharing.

---

## Features

- Choose type of magic: Fire, Water, Earth, Air, Shadow, Light, or Arcane.
- Select difficulty (Easy, Medium, Hard) and the spell's mood.
- Add custom extra ideas for unique spell outputs.
- Beautiful, customizable UI with modern CSS themes.
- Accessible online with no installation needed.

---

## How it Works

- The web app is built with Python and Gradio.
- Each spell is generated by Hugging Face's LLM Inference API.
- [style.py](style.py) controls the entire look and feel (font, card, background).
